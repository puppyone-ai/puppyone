# Skills

Reusable agent instructions for PuppyOne development.

Each `.md` file in this folder is a **Skill** — a set of conventions, rules, and patterns that can be loaded into any AI agent to guide its work.

## How to use

**Claude Code**: Include in `CLAUDE.md` or reference directly:
```
Read /skills/frontend-design.md before writing any UI code.
```

**Cursor**: Copy into `.cursor/rules/` as `.mdc` files.

**Via OpenClaw**: Sync this folder to your local project. Skills are always up to date with the latest conventions.

## Skills index

| File | Purpose |
|------|---------|
| `brand.md` | Product name, positioning, naming conventions |
| `frontend-design.md` | UI design conventions, tokens, component patterns |
