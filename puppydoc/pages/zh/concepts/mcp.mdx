# MCP 协议

AI Agent 如何访问你的 PuppyOne 数据。

## 什么是 MCP？

**MCP (Model Context Protocol)** 是一个开放协议，让 LLM 能够访问外部工具和数据源。支持 MCP 的工具包括：

- Cursor
- Claude Desktop
- 其他 AI 工具

PuppyOne 为你的每个 Table 提供托管的 MCP 服务器。

## 工作原理

```
┌─────────────┐     MCP 协议        ┌─────────────┐
│   AI Agent  │ ◄──────────────────► │  PuppyOne   │
│  (Cursor)   │    tools/prompts    │ MCP Server  │
└─────────────┘                      └─────────────┘
```

1. 你在 Agent 中配置 MCP 端点
2. Agent 发现可用工具（query、create、update 等）
3. Agent 调用工具读写你的上下文数据

## 配置示例

添加到你的 `mcp.json` 或 Cursor 设置：

```json
{
  "mcpServers": {
    "puppyone": {
      "command": "npx -y mcp-remote https://api.puppyone.ai/mcp?api_key=YOUR_KEY"
    }
  }
}
```

## 可用工具

连接后，你的 Agent 可以使用这些工具：

| 工具 | 描述 |
|-----|------|
| `query_data` | 按条件查询数据 |
| `get_all_data` | 获取整个 Table |
| `create` | 添加新条目 |
| `update` | 修改现有条目 |
| `delete` | 删除条目 |

详见 [工具与权限](/zh/concepts/tools)。
