# 自托管部署

在你自己的基础设施上运行 PuppyOne。

## 为什么自托管？

- **数据隐私** — 所有数据保留在你的服务器
- **无限制** — 无速率限制或存储上限
- **可定制** — 按需修改和扩展

## 系统要求

| 组件 | 最低配置 |
|-----|---------|
| Docker | v20+ |
| 内存 | 4GB |
| 存储 | 10GB |
| CPU | 2 核 |

## 快速开始

```bash
git clone https://github.com/puppyone-ai/puppyone.git
cd puppyone
cp .env.example .env
docker-compose up -d
```

打开 `http://localhost:3000` 访问控制台。
