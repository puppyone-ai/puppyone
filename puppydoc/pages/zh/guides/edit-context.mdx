# 编辑上下文

使用可视化编辑器管理你的结构化数据。

## JSON 树形编辑器

PuppyOne 提供可视化的 JSON 数据编辑器：

- **树状视图** — 轻松导航嵌套结构
- **行内编辑** — 点击即可编辑任意值
- **类型感知** — 处理字符串、数字、数组、对象

## 操作

### 添加数据

1. 在目标路径点击 **+** 按钮
2. 选择数据类型（对象、数组、字符串、数字）
3. 输入值

### 编辑数据

1. 点击树中的任意值
2. 行内编辑
3. 自动保存

### 删除数据

1. 右键点击节点
2. 选择 **Delete**
3. 确认删除

## 快捷键

| 快捷键 | 操作 |
|-------|------|
| `Enter` | 确认编辑 |
| `Escape` | 取消编辑 |
| `Tab` | 移动到下一字段 |

## 版本历史

每次编辑都会被追踪：

1. 点击工具栏的 **History**
2. 查看历史版本
3. 点击可恢复任意版本
