# 接入 Agent

将 PuppyOne 集成到你的 AI Agent。

## Cursor

1. 打开 Cursor 设置 → MCP
2. 添加新的 MCP 服务器：

```json
{
  "mcpServers": {
    "puppyone": {
      "command": "npx -y mcp-remote https://api.puppyone.ai/mcp?api_key=YOUR_KEY"
    }
  }
}
```

3. 重启 Cursor
4. 你的 Agent 现在可以访问 PuppyOne 工具了

## Claude Desktop

1. 编辑 `~/Library/Application Support/Claude/claude_desktop_config.json`
2. 添加：

```json
{
  "mcpServers": {
    "puppyone": {
      "command": "npx -y mcp-remote https://api.puppyone.ai/mcp?api_key=YOUR_KEY"
    }
  }
}
```

3. 重启 Claude Desktop

## 自定义 Agent

对于自定义 Agent，使用 REST API：

```bash
# 查询数据
curl -X POST https://api.puppyone.ai/api/v1/tables/{table_id}/query \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"path": "/products", "filter": {"category": "electronics"}}'
```

详见 [API 参考](/zh/api)。

## 获取 API Key

1. 进入 PuppyOne 控制台
2. 点击 **Settings** → **API Keys**
3. 点击 **Create New Key**
4. 复制并安全存储
