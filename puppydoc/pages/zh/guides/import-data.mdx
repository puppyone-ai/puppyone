# 导入数据

从各种来源将数据导入 PuppyOne。

## 支持的来源

| 来源 | 方式 |
|-----|------|
| **文件** | 上传 PDF、DOC、DOCX、图片 |
| **Notion** | OAuth 连接 |
| **Airtable** | OAuth 连接 |
| **GitHub** | OAuth 连接 |
| **Google Sheets** | OAuth 连接 |
| **网页** | URL 爬取 |

## 文件上传

1. 进入你的 Project
2. 点击 **Import** → **Upload Files**
3. 选择文件（PDF、DOC、图片）
4. PuppyOne 自动提取并结构化内容

## 连接器

连接第三方服务：

1. 进入 **Settings** → **Connections**
2. 点击要连接的服务
3. 通过 OAuth 授权
4. 选择要同步的数据

## ETL 流程

当你导入数据时，PuppyOne：

1. **解析** 源文件（图片 OCR、文档 Markdown 提取）
2. **清洗** 内容（去噪、格式标准化）
3. **结构化** 为 JSON（LLM 辅助转换）
4. **存储** 到你的 Table
