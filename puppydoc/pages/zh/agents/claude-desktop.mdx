# Claude Desktop

在 Claude Desktop 中接入 puppyone，让 Claude 查询你的数据。

**接入方式：MCP 协议**

---

## 前提条件

- Claude Desktop（macOS 或 Windows）
- PuppyOne 账号和至少一个 Table

---

## 配置步骤

### Step 1: 获取 MCP 配置

1. 登录 PuppyOne Dashboard
2. 打开你的 Table
3. 点击 **Connect Agent** → **Claude Desktop**
4. 复制生成的配置

### Step 2: 编辑 Claude 配置文件

**macOS**:
```bash
# 打开配置文件
open ~/Library/Application\ Support/Claude/claude_desktop_config.json
```

**Windows**:
```
# 配置文件位置
%APPDATA%\Claude\claude_desktop_config.json
```

如果文件不存在，创建它。

### Step 3: 添加 MCP 配置

```json
{
  "mcpServers": {
    "puppyone": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://api.puppyone.ai/api/v1/mcp/server/sk_live_xxx"]
    }
  }
}
```

### Step 4: 重启 Claude Desktop

完全退出并重新打开 Claude Desktop。

### Step 5: 验证连接

1. 开始新对话
2. 点击对话框旁的 🔌 图标，应该能看到 "puppyone" 服务器
3. 尝试问："帮我查询产品数据"

---

## 配置多个数据源

```json
{
  "mcpServers": {
    "产品目录": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://api.puppyone.ai/api/v1/mcp/server/sk_live_products"]
    },
    "客服FAQ": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://api.puppyone.ai/api/v1/mcp/server/sk_live_faq"]
    }
  }
}
```

## 使用示例

### 查询知识库

```
你：公司的退货政策是什么？

Claude：让我查询一下知识库...
        [调用 query_data]
        
        根据知识库信息，退货政策如下：
        1. 收货后 7 天内可申请退货
        2. 商品需保持原包装
        3. 退款将在 3-5 个工作日到账
```

### 数据分析

```
你：帮我分析一下产品销量数据

Claude：[调用 get_all_data 获取产品数据]
        
        分析结果：
        - 总产品数：150
        - 最畅销：Widget Pro（销量 2,300）
        - 价格区间：¥19.99 - ¥999.99
        - 建议：低于 ¥50 的产品占比较少，可以考虑...
```

## 故障排查

### MCP 服务器没有出现

1. 确认配置文件路径正确
2. 确认 JSON 格式有效
3. 完全重启 Claude Desktop（不只是关闭窗口）

### 错误：npx 未找到

确保已安装 Node.js：

```bash
# 检查 Node.js
node --version

# 如果未安装，安装 Node.js
# macOS: brew install node
# Windows: 从 nodejs.org 下载安装
```

### 连接失败

1. 检查 API Key 是否有效
2. 检查网络连接
3. 尝试在终端手动测试：

```bash
npx -y mcp-remote https://api.puppyone.ai/api/v1/mcp/server/sk_live_xxx --help
```

## 与 Cursor 的区别

| 特性 | Cursor | Claude Desktop |
|------|--------|----------------|
| 代码上下文 | ✅ 自动获取 | ❌ 需手动提供 |
| 文件编辑 | ✅ 直接编辑 | ❌ 只能建议 |
| 对话长度 | 较短 | 较长 |
| 适用场景 | 编程、代码分析 | 文档、分析、问答 |
