## 背后的技术

不同 Agent 的接入方式背后对应不同的技术：

| Agent | 接入方式 | 说明 |
|-------|----------|------|
| Claude Code | 文件同步（双向） | 本地文件夹与云端实时同步，Agent 直接操作文件 |
| OpenClaw | 文件同步（双向） | CLI 守护进程，实时双向同步 |
| Cursor | MCP 协议 | 通过 MCP Server 查询数据 |
| Claude Desktop | MCP 协议 | 通过 MCP Server 查询数据 |
| 沙盒 | 挂载执行 | Context 挂载进隔离沙盒，Agent 可执行代码 |
| 自定义 Agent | REST API | 完全可编程，适合自研工作流 |
