# Google Sheets

从 Google Sheets 导入表格数据到 PuppyOne。

---

## 支持的内容

| 类型 | 支持 | 说明 |
|------|------|------|
| 工作表 (Sheet) | ✅ | 单个或多个 |
| 公式 | ✅ | 导入计算结果 |
| 格式 | ❌ | 只导入数据 |
| 图表 | ❌ | 不支持 |

---

## 设置步骤

### Step 1: 开始导入

1. 打开你的 Table
2. 点击 **Import** → **Google Sheets**

### Step 2: 授权 Google

1. 点击 **Connect Google**
2. 选择 Google 账号并授权
3. 允许 PuppyOne 访问 Google Sheets

### Step 3: 选择电子表格

1. 粘贴 Google Sheets URL，或从列表中选择
2. 选择要导入的工作表（Sheet）
3. 点击 **Import**

---

## 数据结构示例

Google Sheets 会转换为 JSON，第一行作为字段名：

**原始表格：**

| 产品名 | 价格 | 分类 |
|--------|------|------|
| Widget Pro | 99.99 | 电子 |
| Gadget X | 149.99 | 配件 |

**转换结果：**

```json
{
  "sheet_name": "产品列表",
  "rows": [
    {
      "产品名": "Widget Pro",
      "价格": 99.99,
      "分类": "电子"
    },
    {
      "产品名": "Gadget X",
      "价格": 149.99,
      "分类": "配件"
    }
  ]
}
```

---

## 配置选项

| 选项 | 说明 |
|------|------|
| 首行作为标题 | 使用第一行作为 JSON 字段名 |
| 跳过空行 | 忽略没有数据的行 |
| 数字格式化 | 自动识别数字类型 |

---

## 常见问题

### 表格很大，导入会不会很慢？

PuppyOne 使用 Google Sheets API 批量读取，通常几千行的表格可以在几秒内完成。

### 能自动同步吗？

目前需要手动同步。自动同步功能即将推出。
