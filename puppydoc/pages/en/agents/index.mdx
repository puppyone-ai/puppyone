## What's happening under the hood

Each agent connects to PuppyOne differently:

| Agent | Method | How it works |
|-------|--------|--------------|
| Claude Code | File sync (bidirectional) | Local folder syncs with cloud in real time — agent works with files directly |
| OpenClaw | File sync (bidirectional) | CLI daemon keeps local folder and cloud in two-way sync |
| Cursor | MCP protocol | Queries context via MCP Server |
| Claude Desktop | MCP protocol | Queries context via MCP Server |
| Sandbox | Mounted execution | Context mounted into isolated sandbox — agent can run code |
| Custom Agent | REST API | Fully programmable, for custom workflows and scripts |
