# Custom Agent

Use the PuppyOne REST API to integrate with custom agents, scripts, and workflows.

**Connection method: REST API**

---

## Base URL

```
https://api.puppyone.ai/api/v1
```

## Authentication

All requests require a Bearer token:

```
Authorization: Bearer YOUR_API_KEY
```

---

## Core Endpoints

### Query data

```bash
POST /tables/{table_id}/query
```

```bash
curl -X POST https://api.puppyone.ai/api/v1/tables/{table_id}/query \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "filter": {
      "$and": [
        { "status": { "$eq": "active" } },
        { "price": { "$lt": 100 } }
      ]
    },
    "limit": 20
  }'
```

### Get all data

```bash
GET /tables/{table_id}/data
```

### Create entry

```bash
POST /tables/{table_id}/data
```

```bash
curl -X POST https://api.puppyone.ai/api/v1/tables/{table_id}/data \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "name": "New Product", "price": 49.99 }'
```

### Update entry

```bash
PUT /tables/{table_id}/data
```

### Delete entry

```bash
DELETE /tables/{table_id}/data
```

---

## Filter Syntax

Filters follow MongoDB-style syntax:

```json
// Equals
{ "status": { "$eq": "active" } }

// Greater than
{ "price": { "$gt": 50 } }

// In array
{ "category": { "$in": ["electronics", "accessories"] } }

// AND
{ "$and": [ { "status": { "$eq": "active" } }, { "price": { "$lt": 100 } } ] }

// OR
{ "$or": [ { "priority": { "$eq": "urgent" } }, { "priority": { "$eq": "high" } } ] }
```

**Supported operators:** `$eq`, `$ne`, `$gt`, `$gte`, `$lt`, `$lte`, `$in`, `$nin`

## Python Example

```python
import requests

API_KEY = "YOUR_API_KEY"
TABLE_ID = "your-table-id"
BASE_URL = "https://api.puppyone.ai/api/v1"

headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

# Query active products under $100
response = requests.post(
    f"{BASE_URL}/tables/{TABLE_ID}/query",
    headers=headers,
    json={
        "filter": {
            "$and": [
                {"status": {"$eq": "active"}},
                {"price": {"$lt": 100}}
            ]
        }
    }
)

data = response.json()
print(data)
```

## Error Codes

| Code | Meaning |
|------|---------|
| `401` | Invalid or expired API Key |
| `403` | Insufficient permissions (check FLS) |
| `404` | Table or resource not found |
| `422` | Validation error in request body |
| `429` | Rate limit exceeded |
| `500` | Internal server error |
