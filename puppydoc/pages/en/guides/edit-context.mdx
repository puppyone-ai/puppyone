# Edit Context

Manage your structured data with the visual editor.

## JSON Tree Editor

PuppyOne provides a visual editor for your JSON data:

- **Tree view** — Navigate nested structures easily
- **Inline editing** — Click to edit any value
- **Type-aware** — Handles strings, numbers, arrays, objects

## Operations

### Add Data

1. Click the **+** button at the desired path
2. Choose the data type (object, array, string, number)
3. Enter the value

### Edit Data

1. Click on any value in the tree
2. Edit inline
3. Changes auto-save

### Delete Data

1. Right-click on a node
2. Select **Delete**
3. Confirm deletion

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Confirm edit |
| `Escape` | Cancel edit |
| `Tab` | Move to next field |

## Version History

Every edit is tracked:

1. Click **History** in the toolbar
2. View past versions
3. Click to restore any version
