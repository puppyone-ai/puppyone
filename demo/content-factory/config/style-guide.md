# Content Style Guide

## Voice & Tone
- Professional but approachable
- Technical accuracy over simplification
- Use "you" to address the reader directly

## Structure
Every blog post must include:
1. **Title** — concise, under 70 characters
2. **Introduction** — hook + context (2-3 paragraphs)
3. **Body** — organized with H2/H3 headings
4. **Key Takeaways** — bulleted summary
5. **Further Reading** — 2-3 relevant links

## Code Examples
- Always include runnable code snippets
- Use syntax highlighting with language tags
- Add comments for non-obvious logic

## SEO
- Primary keyword in title and first paragraph
- Meta description: 150-160 characters
- At least one image with alt text
