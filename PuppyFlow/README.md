# PuppyFlow (PuppyAgent Frontend Code)

## How to use

First, clone the `PuppyAgent-Engine` repo to your local pc.
```bash
git clone https://github.com/PuppyAgent/PuppyAgent-Engine.git
```

Second, cd into this cloned folder's PuppyFlow subfolder
```bash
cd xxx/PuppyAgent-Engine/PuppyFlow 
# xxx means the local path you put PuppyAgent-Engine into
```


Third, download dependencies
```bash
npm install next
# or try to let system determine what to install
npm install
```

Finally, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Enjoy !!!
