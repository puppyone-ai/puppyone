Add Element - 在数据中添加新元素。

适用内容类型：
- JSON：在数组中追加元素，或在对象中添加新键值对
- Folder：（即将支持）在文件夹中创建新文件

表格：{table_name}
{table_description}

参数：
- elements (List[Dict], 必填): 要添加的元素数组
  - key (str): 元素键名（JSON 对象必须唯一；JSON 数组时可忽略）
  - content (Any): 元素内容，任意 JSON 可序列化的值

行为说明：
- 目标为 Dict 时：按 key 写入；key 必须唯一（不可与现有 key 重复）
- 目标为 List 时：将每个 content 追加到列表末尾；key 会被忽略但仍需提供

使用前提：先用 get_data_schema 了解现有数据结构

示例：
elements = [
    {{"key": "user_001", "content": {{"name": "Alice", "age": 25}}}},
    {{"key": "user_002", "content": {{"name": "Bob", "age": 30}}}}
]

注意：
- 仅适用于 JSON 类型数据
- key 必须唯一，重复的 key 会导致添加失败
