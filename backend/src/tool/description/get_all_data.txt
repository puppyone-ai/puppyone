Get Content - 获取完整内容。

适用内容类型：
- JSON：获取完整 JSON 数据
- Markdown：获取完整文档内容
- Folder：获取文件夹内的文件列表
- Image：获取图片 URL 和元信息

表格：{table_name}
{table_description}

使用场景：
- 需要查看完整数据时使用
- 数据量较小时适用
- 获取文件列表或元信息

返回内容：
- JSON：完整的 JSON 数据
- Markdown：文档文本内容
- Folder：子文件/文件夹列表
- Image：图片 URL 和元数据

注意：数据量大时建议使用 Search 进行检索，避免返回过多数据
