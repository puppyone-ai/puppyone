# Connect your own data

After the first agent write-back works, replace this starter context with your
own data.

Good next steps:

- Upload files or folders from your computer.
- Push an existing Git repo into Puppyone.
- Import a GitHub repo.

The important part is not which path you choose. The important part is that the
agent gets useful context and can write a result back into the same place.

## Suggested second loop

1. Add one real source of context.
2. Give the setup prompt to your coding agent again.
3. Ask it to summarize, clean up, or create a small output file.
4. Check the result and the version history in Puppyone.
