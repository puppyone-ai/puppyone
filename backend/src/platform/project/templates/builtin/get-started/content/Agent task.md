# Agent task

You are inside a PuppyOne starter repo.

Your job is to prove that you can read context from PuppyOne and write a result
back into PuppyOne.

## Instructions

1. Read every file in `Context/`.
2. Create a new Markdown file at `Agent Output/first-run.md`.
3. In that file, write:
   - A 3-bullet summary of what this repo is about.
   - One concrete suggestion for what the user should connect or import next.
   - A short note saying which agent/tool completed the task, if you know.

Do not overwrite this file. Create the output file instead.

## Success condition

The task is complete when `Agent Output/first-run.md` exists and PuppyOne shows
the write in project history.
