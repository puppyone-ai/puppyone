Search - 在内容中进行 AI 语义搜索。

适用内容类型：
- JSON：搜索 JSON 数据中的文本内容
- Markdown：搜索文档内容
- Folder：搜索文件夹内所有文件的内容

表格：{table_name}
{table_description}

参数：
- query: string（必填）自然语言搜索查询
- top_k: integer（可选，默认 5，上限 20）返回结果数量

工作原理：
1. 将查询文本转换为向量表示
2. 在已索引的内容中进行相似度匹配（ANN）
3. 返回最相关的内容片段

输出：
- results: 命中列表，每项包含：
  - score: 相似度分数
  - chunk_text: 匹配的文本片段
  - json_pointer: 内容在原数据中的位置

使用场景：
- 用自然语言查找相关内容
- 在大量数据中快速定位信息
- 语义匹配（不需要精确关键词）

注意：
- 需要先启用 Search 工具对内容建立索引
- 适用于文本内容，不适用于图片
