批量删除指定的键值对。

表格：{table_name}
{table_description}

参数：
- keys (List[str], 必填): 要删除的 key 列表

挂载点类型说明（由 MCP 实例的 json_path 决定）：
- 挂载点为 Dict：keys 是要删除的键名；若某个 key 不存在会失败
- 挂载点为 List：keys 视为下标（如 "0", "1"）；按倒序删除避免下标位移；下标必须在范围内，否则会失败

使用前提：先用 get_data_schema 或 query_data 确认 key 存在

示例：
keys = ["user_001", "user_002"]

注意：
- 删除操作不可逆，请谨慎使用
- 若传入不存在的 key/下标，会导致本次删除失败
